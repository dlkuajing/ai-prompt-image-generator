📝 技术说明

  • 修改内容：
    • 更新了 Gemini API 的提示模板，明确要求生成完整的英文提示词
    • 修改了离线变体生成函数，先翻译基础提示词，再组合维度描述
    • 确保每个变体都是独立完整的，可直接用于 Imagen 4 图片生成
  • 变体格式：
    • 格式：[英文基础内容], [维度描述1], [维度描述2], ...
    • 示例：A man and a woman arguing, dramatic lighting, close-up shot
  • 优点：
    • 避免了中英文混合
    • 变体直接可用，无需额外处理
    • 提高了图片生成的质量和准确性
